# Финальные результаты

В этой папке хранятся готовые результаты работы системы: сгенерированные гипотезы, отчеты и экспортированные данные.

## Содержание

- Список гипотез с обоснованиями и ссылками на источники.
- Дорожные карты проверки гипотез.
- Бизнес-отчеты в формате PDF.
- Экспортированные данные для Jira/YouTrack в формате CSV.
