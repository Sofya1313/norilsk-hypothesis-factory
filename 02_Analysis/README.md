# Аналитические данные

В этой папке сохраняются промежуточные результаты работы интеллектуальной системы, полученные в ходе анализа исходных данных.

## Содержание

Здесь могут находиться:
- Выявленные паттерны потерь по классам крупности.
- Агрегированные данные по минералогическому составу хвостов.
- Структурированные выводы о проблемных зонах технологического процесса.

## Примечание

Все файлы в этой папке создаются автоматически. Предназначены для проверки и валидации экспертами, ручное редактирование не производится.
